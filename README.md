## Commands


ls: Lists files in the RAM disk. Use -l for long format showing permissions and size.

cat: Prints file content to the terminal. Requires read permission.

nano: Text editor for creating and modifying files. Use Tab to save and Esc to quit.

rm: Deletes a file and frees heap memory. Requires write permission.

cp: Copies source file content to a new destination file.

mv: Renames a file or moves it to a new location.

chmod: Changes file permissions using r, w, and x flags.

wc: Displays line, word, and byte counts for a specific file.

grep: Searches for a text pattern in a file and prints matching lines.

hexdump: Shows raw binary data in hex and ASCII columns.

help: Displays a list of all available commands and basic usage.

cls: Clears the terminal screen.

sysinfo: Shows CPU vendor, CPU brand, current date, time, and free heap memory.

uptime: Displays system running time based on the CPU timestamp counter.

theme: Changes color schemes using a number from 0 to 4.

history: Lists the last 20 commands entered in the shell.

echo: Prints the provided text. Expands environment variables starting with a dollar sign.

set: Creates or modifies environment variables using the key equals value format.

env: Lists all currently defined environment variables.

ps: Lists all active and zombie processes with their PIDs and names.

kill: Terminates a process by its process ID.

run: Executes a .c or .tiny script file if it has execute permissions.

pipe: Redirects output from the first command into the second command using the vertical bar.

calc: Opens a command line calculator supporting basic arithmetic and exponents.

ttt: Starts an interactive game of Tic-Tac-Toe.

gui: Switches the kernel into VGA Mode 13h graphical desktop mode with windows.

reboot: Restarts the system by pulsing the keyboard controller reset line.

shutdown: Powers off the system using ACPI commands.
